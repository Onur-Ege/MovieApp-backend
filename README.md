# 🎬 Movie Recommendation API

This is the backend API for a movie recommendation system, built with **Spring Boot**, **Neo4j**. 
It provides movie recommendations using collaborative filtering, graph algorithms, and graph machine learning techniques.

---

## 🚀 Features

- Search movies by title
- Rate movies (1–5 stars)
- Fetch favorite and rated movies
- Fetch popular,disvore, new movies from tmdb
- Fetch movie details
- Fetch personalized movie recommendations


---

## 📦 Tech Stack

| Layer        | Technology                     |
|-------------|---------------------------------|
| Language     | Java (Spring Boot)             |
| Database     | Neo4j (Graph DB + GDS plugin)  |
| External API | TMDb API for movie metadata    |
